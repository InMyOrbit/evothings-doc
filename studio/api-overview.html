API Overview

Evothings Client ships with selected Cordova plugins, which are listed below.

Evothings also provides a number of JavaScript libraries, some of which are built on top of the Cordova plugins, some are for generic use.

Below the API documentation for plugins and libraries are listed.

API documentation for Evothings JavaScript libraries are found at: evothings.com/documentation/libs/

The source code for the JavaScript libraries is found on GitHub: https://github.com/evothings/evothings-examples/tree/master/resources/libs/evothings

Some of the JavaScript libraries are written to communicate with Arduino sketches. The Arduino source code is found here: https://github.com/evothings/evothings-examples/tree/master/resources/arduino
