RedBearLab - Simple Control

This application is a reimplementation of an example application that RedBearLab provides (iOS, Android). The application shows you how to communicate in both directions with your RedBearLab board. In this particular example some hardware is connected the development board.

Source Code

You can browse the source code for this example at the Evothings GitHub repository

The file index.html contains the HTML markup of the app and the app.js contains the logic.

What You Need

This example runs in Evothings Client on Android or iOS, or in any web browser (mobile or desktop).

An iOS device or an Android device with support for Bluetooth 4.0 (which includes BLE) is required. For Android version 4.3 or later is needed. Please note that BLE support on Android is still not fully mature. As a result, you may experience difficulties running this example. If the app stops working, restart the Evothings client app and/or reset Bluetooth on the device.

You also need one of RedBearLabs development boards that has an implementation of the Simple Control example available.

How To Get Up And Running

Follow these steps to get started with this example:

  • Compile and upload the Simple Control application to your RedBearLab board. Since the example is available across many of RedBearLabs products you have to find the correct software on their site.
  • Run the Evothings Workbench on your desktop/laptop computer.
  • Connect to the Workbench from either the Evothings client application.
  • Run the "RedBearLab - Simple Control" example from the Workbench window.
  • Press "Scan" and then connect to your board.
  • Play around with the application.
